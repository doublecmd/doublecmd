Internal commands

Command Description Default
Hot Keys
cm_About Show program "About" information F1
cm_AddPathToCmdLine Place path in command line Ctrl+P
cm_CalculateSpace Calculate occupied space Ctrl+L
cm_ChangeDirToRoot Go to root directory ("/" in GNU/Linux)
cm_CheckSumCalc Calculate file checksum (MD5 or SHA1)
cm_CheckSumVerify Verify MD5 or SHA1 checksum
cm_CompareContents Open compare by contents dialog
cm_ContextMenu Shows context menu for files and folders
cm_Copy Copy items from source to target F5
cm_CopyFullNamesToClip Copy item full names to clipboard Ctrl+Alt+C
cm_CopyNamesToClip Copy item names to clipboard Ctrl+Alt+X
cm_CopySamePanel Copy items in the same directory Shift+F5
cm_CopyToClipboard Copy selected text, file or directory to clipboard Ctrl+C
cm_CutToClipboard Cut selected text, file or directory to clipboard Ctrl+X
cm_Delete Delete selected item to Trash or Recycle Bin F8
cm_DirHistory Invoke dir history drop down menu Ctrl+H
cm_DirHotList Directory hotlist Ctrl+D
cm_DriveContextMenu Shows context menu of a drive. Mount point set at parameters, e.g. (/media/cdrom)
cm_Edit Open file in internal editor F4
cm_EditComment Create or edit comment for file/directory Ctrl+Z
cm_EditNew Create a new file, and open it in the editor (see configuration) Shift+F4
cm_Exchange Swap file panels Ctrl+U
cm_Exit Close Double Commander Alt+X
cm_ExtractFiles Unpack one or more selected archives Alt+F9
cm_FileAssoc Open file associations configuration
cm_FileLinker Combine multiple split files with extension .split
cm_FileProperties Show item properties (size, data, attributes, etc.) Alt+Enter
cm_FileSpliter Split files dialog
cm_FocusCmdLine Change focus to the command line Shift+F2
cm_HardLink Create hard link
cm_HelpIndex Open Double Commander help index page
cm_Keyboard Open keyboard shortcuts help page
cm_LeftEqualRight Show same directory in opposite panel Alt+Z
cm_LeftOpenDrives Open drives list for left panel Alt+F1
cm_MakeDir Create a new directory F7
cm_MarkCurrentExtension Mark files with the same extension as file under cursor
cm_MarkInvert Invert selection
cm_MarkMarkAll Mark all in the panel Ctrl+A
cm_MarkMinus Unselect marked items using a mask
cm_MarkPlus Select items using a mask
cm_MarkUnmarkAll Unselect all marked items F1
cm_MultiRename Open multi rename tool dialog Ctrl+M
cm_NewTab Create a new tab for opened directory Ctrl+T
cm_NextTab Change to next tab Ctrl+Tab
cm_Open Open a file or program under cursor Enter
cm_OpenArchive Try to open a file as an archive Ctrl+PgDown
cm_OpenDirInNewTab Open directory at new tab, but don't switch to it
cm_OpenVFSList Open VFS list (depends on installed plugins)
cm_Options Open configuration dialog
cm_PackFiles Pack items into archive Alt+F5
cm_PanelSpliterPerPos Set the panels width, e.g. 50% — equal size
cm_PasteFromClipboard Paste data from clipboard Ctrl+V
cm_PrevTab Change to previous tab Shift+Ctrl+Tab
cm_QuickSearch Quick search in directory Ctrl+S
cm_Refresh Refresh panels Ctrl+R
cm_RemoveAllTabs Close all tabs
cm_RemoveTab Close active tab Ctrl+W
cm_Rename Rename or move items F6
cm_RenameOnly Rename files in the same directory Shift+F6, F2
cm_ReverseOrder Invert sorting items in the directory
cm_RightOpenDrives Open drives list for right panel Alt+F2
cm_RunTerm Starts a terminal F9
cm_Search Open Search dialog Alt+F7
cm_ShowButtonMenu Create a context menu. Needs parameters
cm_ShowCmdLineHistory Show history of all commands used in the command line Ctrl+7
cm_ShowMenu Open Double Commander's main menu
cm_ShowSysFiles Show hidden and system files
cm_SortByAttr Sort items in file panel by attributes
cm_SortByDate Sort items in file panel by date Ctrl+F5
cm_SortByExt Sort items in file panel by extension Ctrl+F4
cm_SortByName Sort items in file panel by name Ctrl+F3
cm_SortBySize Sort items in file panel by size Ctrl+F6
cm_SymLink Create symlink to file/directory
cm_TargetEqualSource Show same directory in opposite panel Alt+Z
cm_ToggleLockDcaTab Lock current tab, but able to change directory
cm_ToggleLockTab Lock tab
cm_TransferLeft Open directory under cursor in the left panel Ctrl+Left
cm_TransferRight Open directory under cursor in the right panel Ctrl+Right
cm_UnmarkCurrentExtension Unmark items using mask
cm_View Open file in viewer program F3
cm_VisitHomePage Go to Double Commander's home page
cm_Wipe Wipe file Alt+Del
